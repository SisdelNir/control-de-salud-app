# SaludTrack - Control de Salud

Aplicación web para el control de toma de medicamentos diarios con sistema de alertas y recordatorios.

## Características

- Registro de medicamentos con dosis y frecuencia.
- Cronograma diario automatizado.
- Alertas visuales y sonoras.
- Notificaciones del navegador.
- Persistencia de datos local.
- Diseño responsive y premium.

## Tecnologías

- HTML5 Semántico
- CSS3 (Vanilla) con variables y animaciones.
- JavaScript ES6+
- LocalStorage para almacenamiento.
- Web Audio API para alertas.
- Web Notifications API para recordatorios externos.
